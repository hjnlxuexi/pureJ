## 准备

nodeJS：6+

jdk：8+

maven：3+

## 安装配置工具

```shell
#克隆项目
git clone https://github.com/hjnlxuexi/pureJ.git

#进入launcher目录
cd launcher

#安装依赖
npm install

#启动launcher
npm run dev
```

浏览器访问：http://localhost:15910/#/
